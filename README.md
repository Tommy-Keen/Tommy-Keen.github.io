# Blog Workflow

This repository stores the Hexo source for `https://tommy-keen.github.io/`.

## Local writing

Create a new post:

```powershell
npm run new:post -- "Post Title"
```

The post file will be created in `source/_posts/`.

Write in Markdown. If you need LaTeX, use `$...$` for inline math and `$$...$$` for display math.
The homepage excerpt is generated by `hexo-auto-excerpt`. If you want to control the cut point manually for a post, insert `<!-- more -->`.

## Local preview

```powershell
npm run dev
```

Open `http://localhost:4000/`.

If you want a stricter preview from a clean state:

```powershell
npm run preview
```

## Publish

1. Write or edit the post in `source/_posts/`.
2. Run local preview and check `http://localhost:4000/`.
3. Commit your source changes.
4. Push the `master` branch to GitHub.
5. GitHub Actions builds Hexo and deploys the generated `public/` site to GitHub Pages.

Do not commit `public/` or `node_modules/`.
Do not use `hexo d` in this repository.
